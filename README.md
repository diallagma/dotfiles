## Installation

### Make Symlinks
* ln -s .dotfiles/vimrc .vimrc
* ln -s .dotfiles/tmux.conf .tmux.conf
 
### Make Vim Folders 
* mkdir ~/.vim/swap
* mkdir ~/.vim/backup
* mkdir ~/.vim/undo

### Install Vim Plug
* https://github.com/junegunn/vim-plug
* vim ~/.vimrc
* :PlugInstall

### Install Tmux Plugin Manager
* prefix + I
